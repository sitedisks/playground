Carbon Angular Starter was built with offline and progressive web app in mind. It provides manifest.json and service worker (sw.js). In order to simplify development, service worker is not activated by default as this may get in the way of your workflow. If you're actively working on customizing your service worker or doing testing, remember to activate it by uncommeting the script tag content in index.html.
